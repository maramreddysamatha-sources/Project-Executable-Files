Description:
This interactive dashboard provides a visual exploration of UNESCO World Heritage Sites. It allows users to filter data by country, site category (Cultural, Natural, Mixed), and region. It includes map-based visualization, bar charts, and historical trends that help in identifying endangered sites and global distribution patterns.
 Features Highlighted:
* Dynamic geospatial map of heritage sites
* Filterable charts for country-wise and category-wise analysis
* Time-based insights to analyze trends and site listings
